## Notas clave

- 🦈 Julio = temporada tiburones ballena en Holbox — reservar con antelación
- 🏛 Chichén Itzá: llegar antes de las 9h para evitar calor y grupos
- 🦩 Celestún: salida temprana para los flamencos
- 🏨 Mérida: Fiesta Americana · Cancún: Fiesta Americana Villas
- 🚗 La noche del 25 jul en Valladolid permite visitar Chichén Itzá al amanecer y llegar a Cancún esa tarde
